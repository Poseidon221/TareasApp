# TareasApp (borrador)

Aplicación Android en Java con SQLite que permite gestionar tareas hecho como ejercicio.
y ejercicio de clase con el porfesor:
  https://github.com/Poseidon221/aplicacion.git

## Funcionalidades
- Crear tareas
- Listar tareas
- Eliminar tareas
- Persistencia local con SQLite

## Tecnologías
- Java
- SQLite
- RecyclerView
