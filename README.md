# TareasApp

Aplicación Android desarrollada para la gestión de tareas, permitiendo a los usuarios registrar, visualizar, editar y eliminar tareas de manera sencilla. La aplicación utiliza almacenamiento local para mantener la información organizada y accesible.

## Características

- Agregar nuevas tareas.
- Editar tareas existentes.
- Eliminar tareas.
- Visualizar listado de tareas.
- Almacenamiento local de información.
- Interfaz intuitiva y fácil de usar.

## Tecnologías utilizadas

- Java
- Android Studio
- SQLite
- XML

## Uso

1. Iniciar la aplicación.
2. Agregar una nueva tarea.
3. Consultar el listado de tareas registradas.
4. Editar o eliminar tareas cuando sea necesario.
